## 1. News Articles

When you want to post a news article on the website, please provide:

### Required Information:
- **Title** - The headline of your news (e.g., "CICT Wins National Tech Award")
  - Keep it between 3-200 characters
  
- **Summary** - A short description (1-2 sentences)
  - This appears in previews and search results
  - Keep it between 5-300 characters
  
- **Full Content** - The complete article
  - Write as much detail as needed (minimum 10 characters)
  - You can include formatting like bold, italics, bullet points
  
- **Author Name** - Who wrote this article?

### Optional Information:
- **Featured Image** - A main photo for the article
  - Recommended size: 1200 x 630 pixels
  - Accepted formats: JPG, PNG, GIF, WEBP
  - Maximum file size: 5 MB
  
- **Tags** - Keywords to help people find this article
  - Examples: "technology", "awards", "students", "research"
  
- **Publication Date** - When should this go live?
  - If not provided, it will be published immediately

### Example:
```
Title: "CICT Students Win Hackathon"
Summary: "Three CICT students took first place at the National Coding Competition"
Full Content: "Last weekend, three talented students from the College of ICT..."
Author: "Dr. Maria Santos"
Tags: "competition", "students", "achievement"
Image: [Upload photo of winning team]
```

## 2. Events

When you want to create an event on the website, please provide:

### Required Information:
- **Event Name** - What is the event called?
  - Keep it between 2-100 characters
  - Example: "Tech Summit 2025"
  
- **Short Description** - Brief summary (1-2 sentences)
  - Keep it between 10-200 characters
  
- **Full Description** - Complete event details
  - Include what attendees can expect
  - Minimum 10 characters
  
- **Start Date and Time** - When does the event begin?
  - Format: Month/Day/Year, Hour:Minute AM/PM
  - Example: "December 15, 2025, 9:00 AM"
  
- **End Date and Time** - When does the event finish?
  - Must be after the start date
  
- **Location** - Where is the event happening?
  - Example: "Main Auditorium, CICT Building"
  
- **Organizer Name** - Who is organizing this event?

### Optional Information:
- **Maximum Attendees** - How many people can join?
  - Leave blank for unlimited capacity
  - Example: 100
  
- **Event Banner Image** - A promotional image
  - Recommended size: 1920 x 1080 pixels
  - Maximum file size: 5 MB
  
- **Categories** - Type of event
  - Examples: "workshop", "seminar", "networking", "competition"
  
- **Registration Status** - Can people register now?
  - Yes or No

### Example:
```
Event Name: "AI Workshop for Beginners"
Short Description: "Learn the basics of Artificial Intelligence"
Full Description: "Join us for a hands-on workshop where you'll learn..."
Start: December 20, 2025, 2:00 PM
End: December 20, 2025, 5:00 PM
Location: Computer Lab 3, CICT Building
Organizer: ICT Student Forum
Maximum Attendees: 30
Categories: "workshop", "AI", "beginner-friendly"
```

## 3. Announcements

When you need to make an announcement, please provide:

### Required Information:
- **Title** - The announcement headline
  - Keep it between 3-200 characters
  - Example: "Campus Closure Notice"
  
- **Message** - The full announcement text
  - Minimum 10 characters
  - Be clear and specific
  
- **Priority Level** - How urgent is this?
  - **Low** (Blue) - General information
  - **Medium** (Yellow) - Important updates
  - **High** (Orange) - Critical notices
  - **Urgent** (Red) - Emergency alerts
  
- **Category** - What type of announcement?
  - **General** - General campus announcements
  - **Academic** - Classes, exams, grades
  - **Event** - Event-related announcements
  - **Emergency** - Emergency notifications
  
- **Your Name** - Who is making this announcement?

### Optional Information:
- **Target Audience** - Who should see this?
  - Examples: "students", "faculty", "staff", "everyone"
  
- **Expiration Date** - When should this stop showing?
  - Format: Month/Day/Year
  - Example: "December 31, 2025"
  
- **Image** - Optional visual
  - Recommended size: 1200 x 630 pixels

### Example:
```
Title: "Final Exam Schedule Released"
Message: "The final examination schedule for Semester 1 is now available..."
Priority: High
Category: Academic
Author: "Registrar's Office"
Target Audience: "students"
Expiration: January 15, 2026
```

## 4. Organizations

When adding or updating an organization profile, please provide:

### Required Information:
- **Short Name/Acronym** - How is the organization commonly known?
  - Keep it between 2-20 characters
  - Example: "ICT-SF"
  
- **Full Name** - Official organization name
  - Keep it between 5-100 characters
  - Example: "ICT Student Forum"
  
- **Brief Description** - One paragraph summary
  - Keep it between 50-300 characters
  - Example: "The premier student organization for ICT students"
  
- **Detailed Description** - Complete organization overview
  - Keep it between 200-2000 characters
  - Include history, purpose, activities
  
- **Year Established** - When was the organization founded?
  - 4-digit year (e.g., "2018")
  
- **Mission Statement** - What is the organization's mission?
  - Keep it between 50-500 characters
  
- **Vision Statement** - What is the organization's vision?
  - Keep it between 50-500 characters
  
- **Core Values** - What values guide the organization?
  - List 3-10 values
  - Examples: "Innovation", "Collaboration", "Excellence"
  
- **Key Achievements** - Major accomplishments
  - List 3-20 achievements
  - Examples: "Best Student Organization 2023", "Organized 15+ tech workshops"
  
- **Logo** - Organization logo image
  - Recommended size: 512 x 512 pixels (square)
  - Transparent background preferred
  
- **Banner** - Cover/header image
  - Recommended size: 1920 x 600 pixels
  
- **Brand Colors** - Organization's official colors
  - Provide 3 colors (Primary, Secondary, Accent)
  - Can be color names or hex codes (e.g., "Purple" or "#6e29f6")

### Example:
```
Short Name: "CSS"
Full Name: "Computer Science Society"
Brief Description: "A community of passionate computer science students"
Detailed Description: "The Computer Science Society was founded in 2019..."
Established: 2019
Mission: "To foster excellence in computer science education and practice"
Vision: "To be the leading student organization in computing innovation"
Core Values: "Innovation", "Collaboration", "Integrity", "Excellence"
Achievements: 
  - "Best Technical Organization 2023"
  - "Organized 20+ coding workshops"
  - "500+ active members"
Logo: [Upload square logo]
Banner: [Upload wide banner image]
Colors: 
  - Primary: Blue (#2563eb)
  - Secondary: Light Blue (#60a5fa)
  - Accent: Yellow (#fbbf24)
```

## 5. Organization Members

For each member of an organization, please provide:

### Required Information:
- **Full Name** - Member's complete name
  - Keep it between 3-100 characters
  - Example: "Alexandra Chen"
  
- **Position/Role** - What is their role in the organization?
  - Keep it between 3-100 characters
  - Examples: "President", "Vice President", "Secretary", "Member"
  
- **Profile Photo** - Clear headshot
  - Recommended size: 400 x 400 pixels (square)
  - Professional or semi-professional photo
  
- **Biography** - About the member
  - Keep it between 50-500 characters
  - Include year level, program, interests
  - Example: "Senior Computer Science student passionate about AI and machine learning"

### Optional Information:
- **Join Date** - When did they join the organization?
  - Format: Month/Year (e.g., "August 2022")
  
- **Personal Achievements** - Notable accomplishments
  - List 0-20 achievements
  - Examples: "Led 15+ tech workshops", "Dean's Lister for 3 semesters"
  
- **Responsibilities** - Key duties in the organization
  - List 0-10 responsibilities
  - Examples: "Strategic planning", "Event coordination"
  
- **Skills** - Areas of expertise
  - List 0-15 skills
  - Examples: "Leadership", "Python", "Web Development", "Public Speaking"
  
- **Career Timeline** - Major milestones
  - Year and description
  - Example: "2023 - Elected as CSS President"
  
- **Photo Gallery** - Additional photos
  - Multiple images showing member activities
  
- **Social Media Links**
  - LinkedIn profile URL
  - GitHub profile URL
  - Email address

## 6. Images and Photos

### General Guidelines:

**Accepted File Types:**
- JPG/JPEG
- PNG
- GIF
- WEBP

**Maximum File Size:** 5 MB per image

**Recommended Sizes by Type:**

| Image Type | Recommended Size | Notes |
|------------|------------------|-------|
| News Featured Image | 1200 x 630 pixels | Landscape orientation |
| Event Banner | 1920 x 1080 pixels | Wide landscape |
| Organization Logo | 512 x 512 pixels | Square, transparent background |
| Organization Banner | 1920 x 600 pixels | Wide landscape |
| Member Photo | 400 x 400 pixels | Square, headshot |
| Announcement Image | 1200 x 630 pixels | Landscape orientation |

### Tips for Good Images:
- ✅ Use high-quality, clear photos
- ✅ Ensure good lighting
- ✅ Avoid blurry or pixelated images
- ✅ Use relevant images that match your content
- ✅ For logos, use transparent backgrounds when possible
- ❌ Don't use copyrighted images without permission
- ❌ Avoid images with watermarks

## 7. Contact Information

### For User Accounts:

When creating accounts for administrators or members, we need:

**Required:**
- **Email Address** - Must be a valid email
  - Example: "admin@cict.edu"
  
- **First Name** - Person's first name
  - Keep it between 2-50 characters
  
- **Last Name** - Person's last name
  - Keep it between 2-50 characters
  
- **Password** - Secure password
  - Minimum 8 characters
  - Must include: 1 uppercase letter, 1 lowercase letter, 1 number, 1 special character
  - Example: "SecurePass123!"
  
- **Role** - What can this person do?
  - **Full Administrator** - Complete access to everything
  - **Semi Administrator** - Limited admin access
  - **Support Staff** - View and basic operations

### Example:
```
Email: maria.santos@cict.edu
First Name: Maria
Last Name: Santos
Password: Admin@2025!
Role: Full Administrator
```

## ❓ Frequently Asked Questions

### Q: What if I don't have all the information?
**A:** That's okay! Provide what you have. Items marked as "Required" are essential, but "Optional" items can be added later.

### Q: Can I update information after submitting it?
**A:** Yes! All information can be updated at any time through the admin panel.

### Q: What if my image is too large?
**A:** You can use free online tools to resize images:
- [TinyPNG](https://tinypng.com/) - Compress images
- [ResizeImage.net](https://resizeimage.net/) - Resize images
- Or ask the developer for help!

### Q: How do I know what color code to use?
**A:** You can describe the color in words (e.g., "Purple", "Dark Blue") and the developer will help you find the exact code. Or use a color picker tool online.

### Q: Can I write the content in a Word document first?
**A:** Absolutely! Write your content in Word, Google Docs, or any text editor, then copy and paste it when ready.

### Q: What if I make a mistake?
**A:** No problem! Everything can be edited. Just let the developer know what needs to be corrected.
